## atomic-design-template
누구나 vue.js 기반 atomic desigin을 적용하기 쉽도록 template 작성
<br><br>


## 📝 사용언어, 기술스택
- Platform : Web
- Language : Javascript
- Framework : Vue.js(3.0.11),


<strong>Build a vue project</strong>
```cd atomic-design-template
npm install 
npm run serve
```

<strong>Build a storybook</strong>
```cd atomic-design-template
npm run storybook
```
